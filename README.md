# NeuroAge

**Brain Age Gap Technology (BAGT) - Mobile Application**

NeuroAge is a prototype Android application that performs Alzheimer's disease classification and brain age estimation from structural MRI scans. It is based on the BAGT multi-task deep learning framework.

## Download

- **APK Download:** [NeuroAge.apk](NeuroAge.apk)
- **Google Play Testing:** [Join the testing program](https://play.google.com/apps/testing/com.gs.neuroage)

## Features

- MRI volume upload and preprocessing
- Alzheimer's disease classification (CN vs AD)
- Brain age prediction and Brain Age Gap (BAG) computation
- Grad-CAM-based visualization of disease-relevant regions
